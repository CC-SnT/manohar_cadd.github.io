# Manohar, C.S. — Research Portfolio Website

Static research portfolio website built with HTML, CSS and JavaScript.

## Edit
- Update `index.html` for text, links, and publications.
- Update `style.css` for colors and layout.
- Update `assets/` with CV, profile photo, and publication PDFs.

## GitHub Pages Hosting
1. Create a repository named `YOUR-USERNAME.github.io`.
2. Upload all files from this folder into the repository root.
3. Go to Settings → Pages.
4. Choose Deploy from branch → main → root.
5. Open `https://YOUR-USERNAME.github.io`.

## Optional
Add a profile photo as `assets/profile.jpg` and insert it in the hero section.
